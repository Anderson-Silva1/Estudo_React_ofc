REM

A unidade rem (root em) é uma unidade de medida relativa usada no CSS (Cascading Style Sheets) para definir o tamanho de fontes, espaçamentos e outras propriedades de layout. A medida rem é baseada no tamanho da fonte raiz do documento, ou seja, o valor de font-size do elemento <html>.
