Propriedades CSS de Formatação de Texto

1. Propriedades Básicas de Texto

  • color: Define a cor do texto.
  • font-family: Define a família de fontes do texto.
  • font-size: Define o tamanho da fonte.
  • font-weight: Define o peso da fonte (normal, bold, etc.).
  • font-style: Define o estilo da fonte (normal, italic, oblique).

2. Propriedades de Alinhamento

  • text-align: Alinha o texto (left, right, center, justify).
  • text-indent: Define o recuo da primeira linha de um parágrafo.

3. Propriedades de Decoração de Texto

  • text-decoration: Adiciona decorações ao texto, como underline (sublinhado), line-through (tachado) ou overline (sobrelinha).
  • text-transform: Controla a capitalização do texto, com valores como uppercase, lowercase e capitalize.

4. Propriedades Avançadas de Formatação de Texto

  • letter-spacing: Ajusta o espaçamento entre as letras.
  • line-height: Define o espaçamento entre as linhas de texto.
  • word-spacing: Ajusta o espaçamento entre as palavras.
  • text-shadow: Aplica uma sombra ao texto, útil para efeitos visuais.

Exemplo Prático

O código abaixo aplica as principais propriedades de formatação de texto: 

            
p {
    font-size: 18px;
    line-height: 1.6;
    text-align: justify;
    color: #2c3e50;
}
