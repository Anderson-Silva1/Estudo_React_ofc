Posicionamento no CSS

O CSS oferece várias maneiras de posicionar elementos em uma página. A posição dos elementos pode ser controlada usando a propriedade position, que determina como o elemento é posicionado na página.

Posições no CSS

Existem 5 valores principais para a propriedade position:

  • static: O valor padrão. O elemento é posicionado no fluxo normal do documento, ou seja, ele ocupa o espaço de acordo com a sua posição no HTML, e não é afetado pelas propriedades top, right, bottom, ou left.
  • relative: O elemento é posicionado em relação à sua posição original no fluxo do documento. Ou seja, ele pode ser movido em relação à sua posição normal com as propriedades top, right, bottom, ou left.
  • absolute: O elemento é posicionado em relação ao contêiner mais próximo com position: relative, absolute, ou fixed. Se não houver um contêiner posicionado, ele será posicionado em relação ao body (a janela do navegador).
  • fixed: O elemento é posicionado em relação à janela de visualização (viewport). Ele permanece fixo no lugar, mesmo quando o usuário rola a página para cima ou para baixo.
  • sticky: O elemento é posicionado com base no rolar da página. Ele se comporta como um elemento relative até que ele atinja um determinado ponto na tela, onde ele se torna fixo.

Propriedade z-index

O z-index controla a ordem de empilhamento dos elementos. Um elemento com um valor de z-index maior será exibido acima de um elemento com um valor de z-index menor.

Exemplo:
