O que é o CSS Grid?

O display: grid; é uma propriedade do CSS usada para criar layouts em duas dimensões, ou seja, para distribuir e organizar os itens tanto nas colunas quanto nas linhas de um contêiner. Ele permite criar designs complexos de forma simples e eficaz.

Propriedades do Contêiner Grid

O contêiner Grid é o elemento que utiliza a propriedade display: grid;. Aqui estão as principais propriedades que podem ser aplicadas ao contêiner:

  • grid-template-columns: Define o número e o tamanho das colunas do grid.
    • auto: Tamanho automático com base no conteúdo.
    • fr: Unidade flexível, onde 1fr é uma fração do espaço disponível.
    • repeat(): Função para repetir o padrão de colunas (ex: repeat(3, 1fr) cria 3 colunas de tamanho igual).
  • grid-template-rows: Define o número e o tamanho das linhas do grid.
    • auto: Tamanho automático com base no conteúdo.
    • fr: Unidade flexível, similar ao grid-template-columns.
  • grid-template-areas: Permite nomear áreas do grid, facilitando o posicionamento de itens.
    • "header header header": Definindo uma área chamada "header".
    • "content content sidebar": Uma área de conteúdo e uma barra lateral.
  • grid-gap (ou gap): Define o espaço entre as linhas e colunas do grid.
    • gap: 10px;: Cria um espaço de 10px entre as células do grid.
  • justify-items: Alinha os itens ao longo do eixo horizontal.
    • start: Alinha os itens ao início do contêiner.
    • end: Alinha os itens ao final do contêiner.
    • center: Centraliza os itens no eixo horizontal.
    • stretch: Estica os itens para preencher o espaço disponível (padrão).
  • align-items: Alinha os itens ao longo do eixo vertical.
    • start: Alinha os itens ao início do contêiner (no topo).
    • end: Alinha os itens ao final do contêiner (na parte inferior).
    • center: Centraliza os itens no eixo vertical.
    • stretch: Estica os itens para preencher o espaço disponível (padrão).
  • justify-content: Alinha o conteúdo do grid ao longo do eixo horizontal.
    • start: Alinha o conteúdo ao início do contêiner.
    • end: Alinha o conteúdo ao final do contêiner.
    • center: Centraliza o conteúdo no eixo horizontal.
    • space-between: Distribui o conteúdo com espaço igual entre os itens.
    • space-around: Distribui o conteúdo com espaço igual ao redor dos itens.
    • space-evenly: Distribui o conteúdo com espaço igual entre os itens e nas extremidades.
  • align-content: Alinha o conteúdo do grid ao longo do eixo vertical.
    • start: Alinha o conteúdo ao topo do contêiner.
    • end: Alinha o conteúdo ao fundo do contêiner.
    • center: Centraliza o conteúdo no eixo vertical.
    • stretch: Estica o conteúdo para preencher o espaço disponível (padrão).
    • space-between: Distribui o conteúdo com espaço igual entre as linhas.
    • space-around: Distribui o conteúdo com espaço igual ao redor das linhas.

Propriedades dos Itens Grid

Os itens dentro de um contêiner grid podem ser ajustados usando as seguintes propriedades:

  • grid-column-start: Define a linha inicial onde o item começará.
  • grid-column-end: Define a linha final onde o item terminará.
  • grid-row-start: Define a coluna inicial onde o item começará.
  • grid-row-end: Define a coluna final onde o item terminará.
  • grid-column: Atalho para grid-column-start e grid-column-end.
  • grid-row: Atalho para grid-row-start e grid-row-end.
  • grid-area: Permite que um item seja posicionado usando o nome da área definida em grid-template-areas.
