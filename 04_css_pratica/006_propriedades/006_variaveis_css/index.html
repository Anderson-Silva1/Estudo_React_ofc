Exemplo de Uso de Variáveis CSS

As variáveis CSS permitem armazenar valores reutilizáveis, facilitando a manutenção e personalização do estilo de uma página.

O que são Variáveis CSS?

As variáveis CSS, também chamadas de "custom properties", são valores que podem ser definidos uma vez e reutilizados em diferentes partes do código CSS. Elas são muito úteis para manter a consistência e facilitar a manutenção do código, especialmente em projetos grandes.

Como Definir Variáveis CSS?

As variáveis são definidas utilizando o prefixo -- e podem ser colocadas dentro de um seletor, como :root, que é o seletor global para o documento. O valor das variáveis pode ser de qualquer tipo de dado, como cores, tamanhos, fontes, entre outros. 

            
:root {
    --cor-principal: #3498db;
    --cor-secundaria: #2ecc71;
}

Como Usar Variáveis CSS?

Para utilizar uma variável, basta referenciá-la com a função var() no lugar do valor que você deseja substituir. Isso pode ser feito em qualquer parte do CSS onde um valor é esperado. 

            
h1 {
    color: var(--cor-principal);
}

No exemplo acima, o título h1 recebe a cor definida na variável --cor-principal.

Vantagens do Uso de Variáveis CSS

  • Reusabilidade: Uma vez definida, a variável pode ser reutilizada em diferentes partes do código, economizando tempo e evitando a repetição de valores.
  • Facilidade de manutenção: Alterar o valor de uma variável em um único lugar modifica todos os lugares onde ela foi utilizada, tornando a manutenção do código mais eficiente.
  • Maior consistência: Ao usar variáveis para definir cores, tamanhos e outros valores, é fácil manter uma paleta de cores e tamanhos consistente ao longo de todo o projeto.

Exemplo de Mudança de Variável CSS com Media Queries

Você também pode usar media queries para modificar o valor de variáveis CSS dependendo do tamanho da tela do dispositivo. Isso permite criar designs responsivos facilmente. 

            
@media (max-width: 768px) {
    :root {
        --cor-principal: #e74c3c;
    }
}

Com o código acima, a cor da variável --cor-principal será alterada para #e74c3c (vermelho) quando a tela tiver menos de 768px de largura, o que pode ser útil para criar diferentes estilos em dispositivos móveis.
